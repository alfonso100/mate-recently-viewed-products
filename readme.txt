AJAX-powered recently viewed products for WooCommerce. Works with caching. Includes shortcode and block.

== Description ==

This plugin tracks WooCommerce product visits in a cookie and displays them using AJAX. Compatible with full-page caching.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin via WP Admin > Plugins
3. Use the shortcode `[rvpa_recent_products count=5]` or the Gutenberg block.
